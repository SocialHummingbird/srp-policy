# Secret Rare Products Privacy Policy Hosting

This folder is a GitHub Pages-ready static privacy policy package for Secret Rare Products.

## Files

- index.html: default GitHub Pages page.
- privacy-policy.html: stable policy URL target for Google Play.
- privacy-policy.md: source Markdown copy.
- hosting-status.properties: publishability status generated by SRP tooling.
- .nojekyll: disables Jekyll processing.

## Publish With GitHub Pages

1. Create a public GitHub repository for the policy page: $PagesRepoName.
2. Copy every file from this folder into that repository.
3. Commit and push to main.
4. In GitHub, open Settings > Pages.
5. Set Source to Deploy from a branch, Branch main, Folder / (root).
6. Use the GitHub Pages URL ending in /privacy-policy.html as the Play Console privacy policy URL.

Expected Play Console privacy policy URL:

`	ext
https://socialhummingbird.github.io/srp-policy/privacy-policy.html
`

## Publishability

Current status:

`	ext
publishable=true
contactStatus=PASS
contactDetail=Contact section is publishable.
`

Do not use this URL in Google Play until publishable=true.
