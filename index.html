Privacy Policy

Last updated: May 7, 2026

This privacy policy applies to Secret Rare Products, package name `com.tcgmonitor.app`.

Secret Rare Products is an Android app for Pokemon TCG stock monitoring, card scanning, watchlists, guarded checkout review, collection tracking, valuations, backups, and local recovery checks.

Summary

Secret Rare Products does not run an SRP-operated account system or SRP-operated backend. The app stores user-created watchlists, collection records, custom retailer settings, checkout rules, valuation snapshots, backups, and non-secret settings on the user's device.

The app uses network access to fetch retailer pages, Pokemon card catalog data, card images, and optional user-configured AI provider responses. Those third-party services may receive the request data needed to provide their service, such as search terms, URLs, card names, or prompt text.

Data Stored On Device

The app may store the following data locally:

  • Sealed product watch targets.
  • Individual card watch targets.
  • Custom retailer websites, selectors, and preview data.
  • Retailer enablement and checkout settings.
  • Checkout profiles, which may include contact and address details if the user enters them.
  • Guarded checkout review history.
  • Owned-card collection entries.
  • Card valuation snapshots and valuation history.
  • Card catalog, card image, retailer image, and preview caches.
  • Discovery, notification, backup, scanner, and non-secret app settings.
  • User-created backup files when the user exports them or enables scheduled folder mirroring.

API keys are stored locally using Android encrypted preferences and are excluded from app backup exports.

Secret Rare Products disables Android system cloud backup for app data. User-created export files and scheduled folder mirroring are the supported backup paths.

Camera And OCR

The app requests camera permission when the user uses card scanning. Captured card images are used to run on-device OCR and help the user choose a matching catalog card. The OCR flow is user-initiated.

The app does not use the camera for background recording, continuous surveillance, or advertising.

Notifications

The app requests notification permission on Android versions that require it. Notifications are used for restock alerts, discovery review alerts, and foreground monitoring state.

Network Access

The app uses network access for features such as:

  • Checking retailer pages for stock and listing details.
  • Loading Pokemon card catalog data and card images.
  • Loading retailer listing images.
  • Calling optional user-configured AI providers when the user enables AI-assisted review.
  • Opening retailer checkout or product pages when the user chooses to continue manually.

The app does not sell user data.

Backups And Exports

The user can export backup files from the app. Backup files may contain watchlists, custom retailer settings, checkout rules, owned-card records, valuation snapshots, and non-secret settings.

API keys, Android signing keys, and release keystore passwords are excluded from app backup exports.

If the user chooses an external folder for scheduled backup mirroring, Android and the chosen storage provider control access to that folder according to their own settings and policies.

Checkout Automation

Checkout automation is guarded, disabled by default, and user-confirmed. The app is not designed to bypass CAPTCHAs, waiting rooms, bot checks, rate limits, login restrictions, or retailer protections.

Children

Secret Rare Products is not directed to children and does not include an SRP-operated child account system.

Changes

This policy may be updated when app behavior changes. The "Last updated" date will be changed when the policy is revised.
